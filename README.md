# 🐈 Cat Feeder IoT Simulator (PIC24 Architecture Based)

โปรเจกต์จำลองระบบเครื่องให้อาหารแมวอัจฉริยะแบบไร้สาย (Pure Software Simulator) โดยออกแบบโครงสร้างการทำงานเลียนแบบสถาปัตยกรรมไมโครคอนโทรลเลอร์ **PIC24** ร่วมกับเซนเซอร์ตาชั่งน้ำหนักอาหาร ตัวขับเคลื่อนมอเตอร์/Servo และทำงานร่วมกับ **Centralized Web Server (Flask)** แบบ Client-Server API 100%

---

## ติดตั้ง Library
* pip install Flask

## ขั้นตอนการรันระบบจำลอง
* หน้าต่างที่ 1 (รัน Server ส่วนกลาง): python app.py
* python pic24_sim.py

## ✨ ฟังก์ชันเด่นของระบบ (Features)
* **RFID Tag Authentication:** ตรวจสอบสิทธิ์การกินอาหารของแมวรายตัว (รองรับฐานข้อมูลแมว 4 ตัว) และป้องกันการกินอาหารเบิ้ลในหนึ่งวัน
* **Safety Smart Lock System:** เมื่อแมวได้รับสิทธิ์ ประตูเข้าซุ้มจะเปิดออก มอเตอร์จ่ายอาหารทำงาน และสับกลไกล็อคประตูทันทีเพื่อขังแมวไว้ข้างใน ป้องกันแมวตัวอื่นมาแย่งอาหาร
* **Real-time Weight Tracking:** มีระบบลูปดักจับค่าน้ำหนักอาหารในจาน (จำลอง Load Cell + HX711) จะยอมเปิดประตูปล่อยแมวออกก็ต่อเมื่อ **"อาหารหมดจาน (0%)"** แล้วเท่านั้น
* **Live Dashboard Web UI:** หน้าจอแสดงผลสถานะของเครื่องให้อาหาร ปริมาณอาหารในจาน และตารางบันทึกเวลากินล่าสุดของแมวแต่ละตัวแบบสดๆ ผ่านหน้าเว็บ

---

## วิธีทดสอบ Workflow ระบบจำลอง
1 ที่หน้าจอ pic24_sim.py: พิมพ์รหัส Tag ของแมวเพื่อจำลองการทาบการ์ด เช่น TAG_ORANGE_01 (เจ้าส้ม) หรือ TAG_BLACK_02 (เจ้าดำ) แล้วกด Enter

2 สังเกตหน้าเว็บ Dashboard: แถบเกจปริมาณอาหารจะเด้งขึ้นเต็มหลอด 100% ทันที และระบบจะขึ้นสถานะขังแมวไว้

3 จำลองการกินอาหาร: ที่หน้าจอคีย์บอร์ดบอร์ดจำลอง ให้ลองพิมพ์ตัวเลขลดปริมาณอาหารลงเรื่อยๆ เช่นพิมพ์ 60 -> 20 -> และสุดท้ายพิมพ์ 0 แล้วกด Enter

4 สิ้นสุดการทำงาน: เมื่ออาหารเหลือ 0% กลไกจะสั่งเปิดประตูทางออก หน้าเว็บจะบันทึกเวลาการกินล่าสุด และเปลี่ยนสถานะแมวตัวนั้นเป็น "กินแล้ว" ทันที เพื่อป้องกันการแอบกลับมากินซ้ำ

## ลิงก์ยิง API ทดสอบผ่าน Web Browser ตรงๆ

หากไม่รันสคริปต์ฮาร์ดแวร์ สามารถเปิดแท็บใหม่บนเบราว์เซอร์เพื่อยิงคำสั่งเลียนแบบได้:

  * ส่งแมวเข้าเครื่อง (เปิดสิทธิ์): http://localhost:5000/api/check-tag?tag_id=TAG_ORANGE_01

  * แจ้งอาหารหมดจาน (ปล่อยแมวออก/อัปเดตตาราง): http://localhost:5000/api/feed-success?tag_id=TAG_ORANGE_01
